## Repeat grids

- Select (Click and drag) the elements you want to repeat
- On the properties bar click _Repeat Grid_ button
- you can click on the empty space of the grid and resize

If you drag several images over a rectagle that is part of a _Repeat Grid_. It will add a different image on each one.

In a similar fashion, you can drag a text file with several **lines** of text over the text area of a repeat field, and XD will add each line to a different text section.

`CMD-r` Converts a group to a repeat grid

## Adding blur

- Add a background image
- Over the imag create a black rectangle
- Click on the rectangle and on properties select _background blur_

## Masking images

- Create a rectangle
- Drag'n drop on the gray area the image
- Drag the image over the rectangle
- Select both items
- Go to `Object > Mask with shape`
- Select the image agina and resize to required size
