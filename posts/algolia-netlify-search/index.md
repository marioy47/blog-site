# Enable Algolia search using Netlify

A functional blog is asumed

## Intro

Algolia is a hosted search API

All information resides in the Algolia infrastructure

The plugin automatically indexes the content from the website when you deploy

Visit https://algolia.com/netlify

This will take you to the GitHub page

## Features

- Custom domains
- Javascript Execution
- Works with password protected websites
- Hierarchical extraction

## Installation

All the information is in the github page

Its asumed you already have a project

If not, just create a new project from a github repo

Dont forget to specify the build command and the publish directory

**Capture the project name**


Search for the algolia plugin and click on install


Got to the Algolia Dashboard on https://crawler.algolia.com/admin/netlify (or click on _Sign to Algolia with Netlify_ button on the GitHub repo readme)

On algolia click on _Link a new site_

Go to the project an click on the _PLugins_ section
