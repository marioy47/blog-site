# Google Analytics for Developers

There is a demo account for you to play with at https://support.google.com/analytics/answer/6367342?hl=en

`ACCESS DEMO ACCOUNT`

## Create a property

- Account name is whatever
- Property is the name of the website
- Industry category is for reports comparing you to other companies. Work in conjunction with the benchmarking option. Will share anonymous data with you about industry standard

Tracking ID starts with UA (Universal analytics)

If you have multiple propierties under the same account the tracking ID sufix will increament: -1, -2, -3, etc.

## Tracking code

Take a look at the `gtag()` function

There is an alternative to add the code is to use Tag Manager

## Reports

### Realtime

Number of current users

Which pages are active on

Medium is the menas like paid, organic.

Events is if you have event tracking to measure engagement by click on play on a video for instance

Conversions measures goals

### Audience

Get a sense of _who_ is comming to a website: Country, age, sex, browser, language, interest, returning users.
